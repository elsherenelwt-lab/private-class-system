Cara dapet info ini:
1. Sign up gratis di supabase.com
2. Buat New Project (gratis, ~2 menit setup)
3. Di sidebar → SQL Editor → paste isi file supabase-setup.sql → Run
4. Di sidebar → Settings → API → copy Project URL & anon public key ke bawah ini
Penting: Share URL & Key yang sama ke TA. Mereka paste di Settings juga supaya konek ke database yang sama.
